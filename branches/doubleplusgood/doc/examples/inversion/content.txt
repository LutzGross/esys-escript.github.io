data:

- data/QLDWest_grav.nc:
      gravity data, 121x121 cells, Western Queensland, netCDF format
- data/QLDWest_grav.ers, data/QLDWest_grav:
      gravity data, 121x121 cells, Western Queensland, ER Mapper format
- data/QLDWest_mag.nc:
      magnetic data, 121x121 cells, Western Queensland, netCDF format
- data/QLDWest_mag.ers, data/QLDWest_mag:
      magnetic data, 121x121 cells, Western Queensland, ER Mapper format
- data/QLDWest_grav_small.nc:
      gravity data, 40x40 , Western Queensland, netCDF format
- data/QLDWest_grav_small.ers, data/QLDWest_grav_small:
      gravity data, 40x40 , Western Queensland, ER Mapper format
- data/QLDWest_mag_small.nc:
      magnetic data, 40x40, Western Queensland, netCDF format
- data/QLDWest_mag_small.ers, data/QLDWest_mag_small:
      magnetic data, 40x40, Western Queensland, ER Mapper format

scripts:

- create_netcdf.py:
    example script to create netCDF file from python
- plot_netcdf.py:
    script to plot data in netCDF format using matplotlib
- plot_ermapper.py:
    script to plot data in ER Mapper format using matplotlib
- grav_netcdf.py:
    inversion of data/QLDWest_grav_small.nc
    (reads netCDF, creates VTK, Silo and CSV output)
- grav_ermapper.py:
    inversion of data/QLDWest_grav_small.ers
    (reads ER Mapper, creates VTK, Silo and CSV output)
- mag_netcdf.py:
    inversion of data/QLDWest_mag_small.nc
    (reads netCDF, creates VTK, Silo and CSV output)
- mag_ermapper.py:
    inversion of data/QLDWest_mag_small.ers
    (reads ER Mapper, creates VTK, Silo and CSV output)
- gravmag_netcdf.py:
    joint inversion from data QLDWest_grav_small.nc & QLDWest_mag_small.nc
    (reads netCDF, creates VTK, Silo and CSV output)
- gravmag_nodriver.py:
    joint inversion as above, without joint inversion driver (advanced)
    (reads netCDF, creates VTK, Silo and CSV output)
