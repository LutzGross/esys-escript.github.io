=========================
Trilinos Integration
=========================

.. note::
   This chapter is currently being converted from LaTeX to HTML format.
   For the complete content, please refer to the `User Guide PDF <../user/user.pdf>`_.

Overview
========

esys-escript can use the Trilinos solver stack for advanced linear algebra operations.

Features
========

* Advanced iterative solvers
* Preconditioners
* Complex-valued PDEs
* Parallel direct solvers

Configuration
=============

Trilinos support must be enabled at build time. See the Installation Guide for details.
