=====================
Domain Subworlds
=====================

.. note::
   This chapter is currently being converted from LaTeX to HTML format.
   For the complete content, please refer to the `User Guide PDF <../user/user.pdf>`_.

Overview
========

Subworlds allow working with subsets of a domain for specialized computations.

Use Cases
=========

* Region-specific material properties
* Adaptive refinement
* Multi-physics coupling
* Localized computations
