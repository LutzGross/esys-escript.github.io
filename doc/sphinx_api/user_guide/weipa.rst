====================
The weipa Module
====================

.. note::
   This chapter is currently being converted from LaTeX to HTML format.
   For the complete content, please refer to the `User Guide PDF <../user/user.pdf>`_.

Overview
========

weipa provides data export functionality for visualization and post-processing.

Supported Formats
=================

* VTK/VTU files (ParaView, VisIt)
* Silo files
* GOCAD Voxet format

Key Functions
=============

* ``saveVTK()`` - Export to VTK format
* ``saveSilo()`` - Export to Silo format
* ``saveVoxet()`` - Export to GOCAD Voxet format

For detailed API documentation, see :mod:`esys.weipa`.
