The ripley Module

Overview
========

ripley is a fast finite element solver for regular structured meshes in 2D and 3D.

Key Features
============

* Regular structured meshes
* Efficient parallel performance
* Rectangle and Brick domains
* GPU support (experimental)
* Binary grid I/O

For detailed API documentation, see :mod:`esys.ripley`.
