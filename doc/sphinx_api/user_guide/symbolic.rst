===========================
Symbolic Mathematics
===========================

.. note::
   This chapter is currently being converted from LaTeX to HTML format.
   For the complete content, please refer to the `User Guide PDF <../user/user.pdf>`_.

Overview
========

esys-escript provides integration with SymPy for symbolic mathematics operations.

Features
========

* Symbolic differentiation
* Expression simplification
* Code generation
* Integration with Data objects

For detailed API documentation, see :mod:`esys.escript.symbolic`.
