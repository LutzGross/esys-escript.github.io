====================
The finley Module
====================

.. note::
   This chapter is currently being converted from LaTeX to HTML format.
   For the complete content, please refer to the `User Guide PDF <../user/user.pdf>`_.

Overview
========

finley is a finite element solver for unstructured meshes supporting triangles, quadrilaterals, tetrahedra, and hexahedra.

Key Features
============

* Unstructured mesh support
* Multiple element types
* Mesh file import
* Contact elements
* Macro elements

For detailed API documentation, see :mod:`esys.finley`.
